# Website

This website is built using [Astro](https://astro.build/) with the [Starlight](https://starlight.astro.build/) documentation theme.

## Installation

```bash
npm install
```

## Local Development

```bash
npm run start
```

This command starts a local development server at `localhost:4321`. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `dist` directory and can be served using any static hosting service.

## Preview

```bash
npm run preview
```
